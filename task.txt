# Challenge: E-commerce store using laravel with service repository pattern

Features
- Simple role based authentication [ Admin, User ]
  
### Admin Role
- Manage Categories [ Create, Update, Delete ]
- Manage Products By Category [ Create, Update, Delete ]

### User Role
- View Products By Category [ View ]
- Add To Cart & Checkouts [ Can add multiple products to cart and checkout ]
- View Orders [ View ]
